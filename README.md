# AURA — Affective UI Research Assistant

> Real-time emotion-adaptive web interface | HCI Research Project | 2026

## Overview
AURA is a research prototype that detects the user's facial emotion 
in real time via webcam and dynamically adapts the entire UI — 
colour palette, typography, layout density, and content tone — 
to match their emotional state.

## Features
- Real-time face detection using OpenCV
- 7 adaptive emotional themes (Happy, Sad, Angry, Fear, Surprise, Disgust, Neutral)
- Live emotion confidence scores and timeline
- Flask backend with REST API
- Research-grade dark UI with professional branding

## Tech Stack
- Python, Flask, OpenCV
- HTML, CSS, JavaScript
- Web Speech API

## How to Run
pip install -r requirements.txt
python app.py
Open http://127.0.0.1:5000 in Chrome

## Research Context
Built as part of HCI research internship application to 
Centre for Human-Computer Interaction (CHCI).
Explores affective computing and emotion-aware adaptive interfaces.
